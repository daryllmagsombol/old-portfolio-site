# Daryll Joshua Magsombol — Portfolio (2018)

My personal portfolio website from back in 2018. A static single-page site showcasing my early projects and before I got into work in Web Development field.

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3 / Sass** — Custom styles with responsive breakpoints
- **JavaScript (jQuery)** — DOM manipulation, smooth scrolling, nav interactions
- **Skel.js** — Responsive grid and breakpoint framework
- **Font Awesome 4** — Icon set

Built on the [**Prologue**](https://html5up.net/prologue) template by [HTML5 UP](https://html5up.net/) (ajlkn).

## Getting Started

This is a fully static site — no build tools, no dependencies to install. Just open `index.html` in any browser:

```bash
open index.html
```

Or serve it locally:

```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Then visit `http://localhost:8000`.
